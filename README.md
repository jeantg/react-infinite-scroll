## Installation

To install, you can use [npm](https://npmjs.org/):

    $ npm install --save react-infinite-scroll

To install, you can use [yarn](https://https://yarnpkg.com/):

    $ yarn add react-infinite-scroll

| -------- | --------- | ------- | ------------- |
